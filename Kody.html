Jak pisać w kodach HTML?

HTML (HyperText Markup Language) to język znaczników, który jest używany do tworzenia stron internetowych. Poniżej przedstawiamy podstawowe informacje o tym, jak pisać w kodach HTML.

Podstawowe elementy HTML

  • <html>: element główny dokumentu HTML
  • <head>: sekcja nagłówkowa dokumentu, zawiera informacje o dokumencie
  • <title>: tytuł strony internetowej
  • <body>: sekcja treści dokumentu
  • <h1>, <h2>, <h3>, itd.: nagłówki różnych poziomów
  • <p>: akapit tekstu
  • <img>: obrazek
  • <a>: link

Tworzenie struktury strony

  1. Zacznij od elementu <html>, który jest elementem głównym dokumentu HTML.
  2. Wewnątrz <html> dodaj sekcję <head>, która zawiera informacje o dokumencie, takie jak tytuł strony.
  3. W sekcji <head> dodaj element <title>, który określa tytuł strony internetowej.
  4. Po sekcji <head> dodaj sekcję <body>, która zawiera treść dokumentu.

Dodawanie treści

  • W sekcji <body> dodaj nagłówki różnych poziomów za pomocą elementów <h1>, <h2>, <h3>, itd.
  • Dodaj akapity tekstu za pomocą elementu <p>.
  • Dodaj obrazki za pomocą elementu <img>.
  • Dodaj linki za pomocą elementu <a>.

Przykład


<html>
<head>
	<title>Moja strona internetowa</title>
</head>
<body>
	<h1>Witaj na mojej stronie!</h1>
	<p>To jest moja pierwsza strona internetowa.</p>
	<img src="obrazek.jpg" alt="Obrazek">
	<a href="https://www.example.com">Link do innej strony</a>
</body>
</html>
